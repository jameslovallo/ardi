Ramidus

Ramidus is a Single Page Application (SPA) template for the modern web. Built with and for custom elements, Ramidus is a template that provides familiar conventions and a tiny bit of tooling to provide the best UX and DX possible.

Features

  • Familiar app structure, similar to Next or Nuxt.
  • Global layout is shared across multiple pages.
  • Vanilla folder structure and routing.
  • SPA navigation with <app-root> and <app-link>.
  • Write pages in markdown with <mark-down>.
  • No building or tooling required.
