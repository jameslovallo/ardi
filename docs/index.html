Ardi

Welcome to the Weightless Web. Ardi is a tiny (but fierce) web component framework, weighing just 3kb compressed.

Our Mission

  1. Make it easy to create powerful web components that work everywhere, including React, Vue, Svelte, Astro, etc.
  2. Provide kid-friendly DX and great performance without virtual dom, JSX, or compilers.
